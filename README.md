## ğŸš€ Cara Menjalankan Simulasi

Berikut adalah panduan singkat untuk membuka dan menjalankan file-file simulasi di arsip ini.

### 1. Simulasi Excel (.xlsx)

1.  Pastikan Anda memiliki **Microsoft Excel** (versi 2010 ke atas direkomendasikan) atau perangkat lunak spreadsheet yang kompatibel (seperti Google Sheets atau LibreOffice Calc).
2.  Buka file `.xlsx` yang relevan.
3.  Model simulasi biasanya dapat dijalankan ulang dengan menekan tombol `F9` (untuk menghitung ulang formula dengan nilai acak baru) atau dengan mengubah parameter input yang disediakan.

### 2. Simulasi JaamSim (.cfg)

1.  **Unduh JaamSim**: Anda memerlukan perangkat lunak JaamSim untuk membuka file `.cfg`. Unduh versi terbaru dari [situs web resmi JaamSim](http://jaamsim.com/downloads.html).
2.  **Instalasi**: JaamSim adalah aplikasi portabel berbasis Java. Pastikan Anda memiliki Java (JRE) terinstal di komputer Anda. Cukup jalankan file `JaamSim.jar`.
3.  **Buka Model**:
    * Buka aplikasi JaamSim.
    * Pilih `File > Open` dari menu bar.
    * Arahkan ke file `.cfg` yang ingin Anda buka dari repositori ini.
4.  **Jalankan Simulasi**:
    * Setelah model dimuat, Anda dapat menekan tombol **Play** (â–¶) di *toolbar* untuk memulai simulasi.
    * Anda dapat mengatur kecepatan simulasi atau menjalankannya langkah demi langkah.
